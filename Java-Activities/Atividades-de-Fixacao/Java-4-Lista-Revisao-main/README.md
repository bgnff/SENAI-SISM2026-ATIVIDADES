Este projeto foi desenvolvido em sala de aula com o objetivo de aplicar conceitos fundamentais da linguagem Java, com foco em lógica de programação e controle de fluxo.

Foram utilizados operadores lógicos, estruturas condicionais (if, else), estruturas de repetição (while, for), operador ternário e operações com módulo (%), demonstrando na prática o funcionamento dessas construções no desenvolvimento de algoritmos.

O código foi estruturado de forma simples e objetiva, visando reforçar o aprendizado e a compreensão dos principais recursos da linguagem.
