Exercícios com SE e MOD

1. Par ou Ímpar: Leia um número inteiro e, utilizando o operador MOD,
informe se o número é PAR ou ÍMPAR.

2. Aprovação Simples: Leia duas notas de um aluno, calcule a média e
informe se ele está APROVADO (média $\ge 7$) ou REPROVADO (média $<
7$).

3. Múltiplo de 5: Peça para o usuário digitar um número e informe se ele é um
múltiplo de 5 (Dica: o resto da divisão por 5 deve ser zero).

4. Maioridade Penal: Leia o ano de nascimento de uma pessoa e o ano atual.
Calcule a idade e informe se ela já é maior de idade ou não.

5. Divisibilidade Dupla: Leia um número e informe se ele é divisível por 2 e
por 3 ao mesmo tempo (utilize dois testes com MOD).

6. Comparação de Valores: Leia dois números inteiros diferentes e exiba
qual deles é o maior.

7. Positivo ou Negativo: Leia um número e diga se ele é Positivo, Negativo
ou Zero.

8. Calculadora de Resto: Leia dois números e verifique se o primeiro é
divisivel pelo segundo. Se o resto for zero, exiba "Divisão Exata", caso
contrário, exiba "O resto da divisão é: X".

9. Desconto por Volume: Um software de vendas dá 10% de desconto se o
valor da compra for maior que R$ 100,00. Leia o valor total e exiba quanto o
cliente deve pagar (se o desconto se aplicar ou não).

10.Verificação de Login: Crie um programa que peça uma senha ao usuário.
Se a senha for "1234", exiba "Acesso Permitido". Caso contrário, exiba
"Acesso Negado".
